### 3. TimeFM & Temporal Foundation Models for Market Prediction

**What is TimeFM?**
Google Research's **200M parameter time-series foundation model**, pretrained on 100 billion real-world time points. Think "GPT for time series" — it learns universal temporal patterns (seasonality, trends, volatility clustering) and can do **zero-shot forecasting**.

**Key Capabilities (v2.5, Sept 2025):**
- Context length: up to **16,384 points** (long-term pattern capture)
- Horizon: up to 1,000 steps ahead
- Continuous quantile forecasts (full distribution, not just point estimates)
- Supports covariates via XReg
- Fine-tuning with LoRA available

**Financial Application Findings:**
- Off-the-shelf TimeFM performs **poorly** on financial returns (directional accuracy ~51%, out-of-sample R² near zero)
- **BUT** when fine-tuned on financial data (100M+ time points, hourly+daily granularity), it outperforms baselines in:
  - Returns
  - Sharpe ratio
  - Maximum drawdown
  - Trading cost efficiency
- Best results come from **continual pre-training** on domain-specific data, not zero-shot
- Directional accuracy improves significantly with fine-tuning

**Practical Implementation:**
```python

2. Use TimeFM to forecast regime probabilities (bull/bear/sideways)
3. Combine with geospatial data (AlphaEarth) for commodity/sector tilts
4. Generate position sizing based on forecast confidence intervals
5. Walk-forward validation mandatory — no look-ahead bias
```
